Avant de pouvoir utiliser ces macros, il faut copier, les dossiers "css", 
"scripts" et "xsl" dans le dossier "TXM" du répertoire utilsateur. Ces dossiers existent déjà, 
vous devez donc accepter la fusion et l'écrasement du fichir xmlLoader.groovy

Par système, le chemin du dossier TXM ressemble à :
- Windows : C:\Users\<identifiant>\TXM ou C:\Documents and Settings\<identifiant>\TXM (Windows XP)
- Mac OS X : /Users/<identifiant>/TXM
- Linux : /home/<identifiant>/TXM

Plus d'information quant à l'utilisation des macros : 
https://groupes.renater.fr/wiki/txm-users/public/macros
